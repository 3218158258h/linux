
# 条件变量
#### 一般流程
- 定义(`pthread_cond_t cond`)：定义一个条件变量`cond`
- 初始化（`pthread_cond_init`）：创建条件变量并初始化。
- 等待（`pthread_cond_wait` / `pthread_cond_timedwait`）：持有互斥锁并等待条件满足（后者带超时）
- 唤醒（`pthread_cond_signal` / `pthread_cond_broadcast`）：唤醒一个 / 所有等待该条件变量的线程
- 销毁（`pthread_cond_destroy`）：清理条件变量资源（静态初始化不需要）

---

## 1-1. 静态初始化
```c
static pthread_cond_t cond = PTHREAD_COND_INITIALIZER;
```
- **直接定义并初始化**，无需手动销毁
- 全局/静态变量推荐使用，最简单、最安全

---

## 1-2. 动态初始化
`pthread_cond_init`负责为条件变量分配资源并设置属性
```c
#include <pthread.h>
int pthread_cond_init(pthread_cond_t *restrict cond, const pthread_condattr_t *restrict attr);
```
- **必须销毁**：动态初始化必须用 `pthread_cond_destroy` 释放资源

##### 参数解析
- `cond`：指向 `pthread_cond_t` 变量的指针
- `attr`：条件变量属性，`NULL` 表示使用**默认属性**

##### 返回值
- 成功：返回 `0`
- 失败：返回非零错误码

##### 默认属性初始化（最常用）
```c
// 定义条件变量
pthread_cond_t cond;

// 动态初始化（默认属性）
if (pthread_cond_init(&cond, NULL) != 0) {
    perror("cond init failed");
    exit(EXIT_FAILURE);
}

// 使用条件变量...

// 必须销毁
pthread_cond_destroy(&cond);
```

---

## 2-1. 阻塞等待（无限等待）
`pthread_cond_wait` 是条件变量**核心等待函数**，必须**配合互斥锁**使用。
```c
#include <pthread.h>
int pthread_cond_wait(pthread_cond_t *restrict cond, pthread_mutex_t *restrict mutex);
```

##### 核心行为（必须背）
1. **自动解锁 mutex**
2. **线程阻塞休眠**
3. **被唤醒后自动重新加锁 mutex**
4. 返回时，线程**一定持有锁**

##### 返回值
- 成功被唤醒：返回 `0`
- 失败：返回错误码

##### 使用模板（标准写法）
```c
pthread_mutex_lock(&mutex);

// 必须用 while，防止虚假唤醒
while (条件 == 不满足) {
    pthread_cond_wait(&cond, &mutex);
}

// 条件满足 → 处理业务

pthread_mutex_unlock(&mutex);
```

---

## 2-2. 超时等待（嵌入式高频使用）
`pthread_cond_timedwait` 等待指定时间，超时自动返回。
```c
#include <pthread.h>
#include <time.h>

int pthread_cond_timedwait(
    pthread_cond_t *restrict cond,
    pthread_mutex_t *restrict mutex,
    const struct timespec *restrict abstime
);
```

##### 参数说明
- `abstime`：**绝对时间**（不是相对秒数）
- 超时后函数自动返回，不会一直阻塞

##### 超时时间设置示例（等待3秒）
```c
struct timespec ts;
clock_gettime(CLOCK_REALTIME, &ts);  // 获取当前时间
ts.tv_sec += 3;                      // 超时 = 当前时间 + 3秒

int ret = pthread_cond_timedwait(&cond, &mutex, &ts);
if (ret == ETIMEDOUT) {
    // 超时处理
}
```

##### 返回值
- `0`：被正常唤醒
- `ETIMEDOUT`：超时
- 其他：错误

---

## 3-1. 唤醒一个等待线程
`pthread_cond_signal` 唤醒**任意一个**正在等待的线程。
```c
#include <pthread.h>
int pthread_cond_signal(pthread_cond_t *cond);
```

##### 使用规则
- 必须**先修改条件**，再唤醒
- 必须**在持有锁**时调用

##### 示例
```c
pthread_mutex_lock(&mutex);
has_data = 1;               // 修改条件
pthread_cond_signal(&cond); // 唤醒一个
pthread_mutex_unlock(&mutex);
```

---

## 3-2. 唤醒所有等待线程
`pthread_cond_broadcast` 唤醒**全部**等待线程。
```c
#include <pthread.h>
int pthread_cond_broadcast(pthread_cond_t *cond);
```

##### 适用场景
- 多个线程等待同一个条件
- 状态变更后需要全部唤醒

---

## 4. 销毁条件变量
`pthread_cond_destroy` 销毁动态初始化的条件变量，释放内核资源。
```c
#include <pthread.h>
int pthread_cond_destroy(pthread_cond_t *cond);
```

##### 功能与行为
- 只能销毁**未被等待**的条件变量
- 销毁后不可再使用，除非重新初始化
- 静态初始化（`PTHREAD_COND_INITIALIZER`）**不需要销毁**

##### 返回值
- 成功：`0`
- 失败：非零错误码（`EBUSY` 表示有线程正在等待）

---

# 最关键注意事项
1. **必须配合互斥锁使用**
2. `wait` 会**自动解锁 + 唤醒后自动加锁**
3. 等待必须用 **while()**，不能用 if()（防虚假唤醒）
4. 先**修改条件**，再**调用唤醒**
5. 超时等待用 `timedwait`，传入**绝对时间**


### 例程
```c
#include <stdio.h>
#include <pthread.h>
#include <unistd.h>
#include <stdlib.h>

// --------------------- 条件变量 + 互斥锁 ---------------------
pthread_cond_t cond = PTHREAD_COND_INITIALIZER;   // 静态初始化
pthread_mutex_t mutex = PTHREAD_MUTEX_INITIALIZER; // 静态初始化

int has_data = 0; // 共享条件：0=无数据 1=有数据

// --------------------- 消费者线程：等待数据 ---------------------
void *consumer_thread(void *arg)
{
    while (1) {
        // 1. 加锁
        pthread_mutex_lock(&mutex);

        // 2. 等待条件（必须用 while）
        while (has_data == 0) {
            printf("[消费者] 无数据，等待中...\n");
            pthread_cond_wait(&cond, &mutex); // 阻塞 + 自动解锁 唤醒后自动加锁
        }

        // 3. 被唤醒，条件满足
        printf("[消费者] 收到数据，处理完成！\n");
        has_data = 0; // 清空数据

        // 4. 解锁
        pthread_mutex_unlock(&mutex);

        sleep(1);
    }
    return NULL;
}

// --------------------- 生产者线程：产生数据 ---------------------
void *producer_thread(void *arg)
{
    while (1) {
        sleep(2); // 每2秒生产一次数据

        // 1. 加锁
        pthread_mutex_lock(&mutex);

        // 2. 修改条件
        has_data = 1;
        printf("[生产者] 数据已产生，通知消费者...\n");

        // 3. 唤醒等待的线程
        pthread_cond_signal(&cond);

        // 4. 解锁
        pthread_mutex_unlock(&mutex);
    }
    return NULL;
}

// --------------------- 主函数 ---------------------
int main()
{
    pthread_t consumer, producer;

    // 创建线程
    pthread_create(&consumer, NULL, consumer_thread, NULL);
    pthread_create(&producer, NULL, producer_thread, NULL);

    // 等待线程运行
    pthread_join(consumer, NULL);
    pthread_join(producer, NULL);

    // 销毁（静态初始化可省略，这里演示规范写法）
    pthread_cond_destroy(&cond);
    pthread_mutex_destroy(&mutex);

    return 0;
}
```
